## 👨‍💻 Professional Profile

I am a Network Infrastructure and Software Developer currently pursuing my education in **Computer Network & Telecommunication Engineering (TJKT)** at **SMKN 1 Majalengka**. Known to my peers as **Kantee**, I build and operate under the digital brand **ktexucoder**. I specialize in bridging the gap between hardware infrastructure (networking, routing, server administration) and software development (backend scripting, database architecture, and web apps).

My technical journey began with game server administration, leading me to architect and maintain **5+ high-traffic FiveM game servers**. This background uniquely positioned me to master Linux systems administration, performance tuning, and low-latency scripting under real-world conditions.

---

## 💼 Areas of Expertise

### 🌐 Network Engineering & System Administration
* **Enterprise Routing & Switching**: Configuring dynamic routing (OSPF, RIP), VLAN segmentation, bandwidth allocation, and firewall policies on **MikroTik RouterOS** and **Cisco IOS**.
* **Systems Administration**: Deployment and management of production **Linux (Ubuntu Server, Debian)** and Windows Server environments.
* **Virtualization & Cloud Infrastructure**: Setting up bare-metal hypervisors and virtual machines using **Proxmox VE**, VMware ESXi, and VirtualBox.
* **Web Services & Security**: Hosting secure servers with **Nginx** reverse proxies, Apache, SSL/TLS configurations, and server hardening.

### 💻 Full-Stack Development & Scripting
* **Backend Systems**: Designing reliable, type-safe REST APIs and microservices using **Go (Golang)**, **Node.js/TypeScript**, and **Laravel**.
* **Frontend Engineering**: Developing responsive, user-centric interfaces utilizing **React**, **Svelte**, **Tailwind CSS**, and modern build tools.
* **Database Management & BaaS**: Structuring performant relational and non-relational databases using **PostgreSQL**, **MySQL/MariaDB**, **MongoDB**, and **Supabase**.
* **FiveM Architecture**: Creating custom gameplay scripts via **Lua**, server-side profiling to minimize CPU latency, and designing rich in-game interfaces via Web NUI (HTML/CSS/JS).
